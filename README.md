
# JobHub India

Welcome to this project! This repository contains a modern React application built with Vite, TypeScript, and Tailwind CSS, designed with best practices and powered by shadcn-ui.

---

## 🚀 Getting Started

Follow these steps to get your development environment up and running locally.

### Prerequisites

- Node.js and npm installed (We recommend using [nvm](https://github.com/nvm-sh/nvm#installing-and-updating) to manage your Node versions)
- Git installed

### Installation & Running Locally

```bash
# 1. Clone the repository
git clone <YOUR_GIT_URL>

# 2. Change directory to the project folder
cd <YOUR_PROJECT_NAME>

# 3. Install dependencies
npm install

# 4. Start the development server with hot reload
npm run dev
````

Open your browser at [http://localhost:8080](http://localhost:8080) to see the app running.

## 🛠️ Technologies Used

This project leverages a modern, efficient stack:

* **Vite** – Lightning fast build tool and dev server
* **React** – Popular UI library for building interfaces
* **TypeScript** – Strongly typed JavaScript superset for safer code
* **shadcn-ui** – A set of accessible, reusable UI components
* **Tailwind CSS** – Utility-first CSS framework for rapid styling
